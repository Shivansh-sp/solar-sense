## ğŸš€ Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Shivansh-sp/solar-sense.git
   cd solar-sense
   ```

2. **Install dependencies**
   ```bash
   # Frontend
   cd frontend
   npm install
   
   # Backend

   npm install
   ```

3. **Start development servers**
   ```bash
   # Frontend (Terminal 1)
   cd frontend
   npm run dev

   cd backend
   npm start
   ```

4. **Open your browser**
